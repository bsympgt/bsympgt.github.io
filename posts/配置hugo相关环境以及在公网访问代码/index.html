配置hugo相关环境以及在公网访问代码

杨俊浩
 2022-12-14  238 words   One minute 
Contents

下载主题

git clone +地址

新建一个文章文本

hugo new site mysite

cd mysite

hugo new posts/first_post.md

测试是否连接成功

hugo serve - -buildDrafts

在网页上运行

将本地文件push到github里面

echo “# Linux” » README.md git init git add README.md git commit -m “first commit” git branch -M main git remote add origin https://github.com/Zzkyou/Linux.git git push -u origin main

在public目录下执行这条命令

hugo –baseUrl="https://bsympgt.github.io/” –buildDrafts

然后回到mysite目录执行以下命令将本地的内容push到github上

cd public

git init

git add .

git commit -m “ggg”

git branch -M master

git remote add origin git@github.com:bsympgt/bsympgt.github.io.git

git push -u origin master

在公网上运行bsympgt.github.io
